# TaskPro Frontend Application

A sophisticated Kanban-style task management application built with React and Redux, focused on performance, responsive design, and exceptional user experience.

## ✨ Features
- **Kanban Board:** Intuitive interface for creating boards, columns, and cards with a smooth drag-and-drop experience.
- **Dynamic Themes:** Toggle between "Light", "Dark", and "Violet" modes with a single click.
- **User Profiles:** Edit personal information, upload custom avatars, and manage account settings.
- **Board Customization:** Choose from various background patterns and icons for each board.
- **Task Management:** Set priorities (Low, Medium, High), deadlines, and move cards between columns seamlessly.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using styled-components.
- **Protected Routes:** Secure navigation using Private and Restricted routing logic.

## 🚀 Technologies Used
- **Frontend Framework:** React (with functional components and hooks)
- **State Management:** Redux Toolkit & Redux Persist
- **Styling:** Styled-components & Emotion
- **Navigation:** React Router DOM
- **Authentication:** JWT-based auth with token refresh intercepts
- **Drag and Drop:** React Beautiful DnD
- **UI Components:** Material UI (MUI) icons & React Icons
- **Form Management:** Formik & Yup (validation)
- **Date Picking:** React Datepicker
- **Feedback:** React Toastify (notifications) & Loader Spinners

## 🛠 Setup Instructions
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rudopulos/TaskPro_FrontEnd-Public.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Variables:**
    -   Copy `.env.example` to a new file named `.env`.
    -   Set `REACT_APP_API_URL` to point to your running TaskPro Backend.
4.  **Run the application:**
    ```bash
    npm start
    ```
- The application will be available at `http://localhost:3000`.
